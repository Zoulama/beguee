### What is this repository for? ###

Aller à la racine de dossier de travail:

git clone https://zoulama-d@bitbucket.org/beguee/beguee.git

### How do I get set up? ###

* cd beguee
* cd application 
* cp .env.example .env
* Revenir à la racine beguee
* make build
* make start
* lancer l'application sur http://localhost:8085

### stop project ###
* cd beguee
* make stop
* make clean
